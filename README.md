# Customer Account Data Pipeline (ADF)

## Overview
Built an end-to-end ETL pipeline using Azure Data Factory to process customer datasets.

## Architecture
Bronze → Silver → Gold architecture using ADLS Gen2

## Tools Used
- Azure Data Factory
- ADLS Gen2
- Azure SQL
- Power BI

## Key Features
- Data ingestion from CSV files
- Data transformation using Dataflows
- SCD Type 1 & Type 2 implementation
- Parameterized pipelines
